# Float

レスポンシブ フロート ユーティリティを使用して、任意のブレークポイントにカスタム フロートを適用します。

<entry-ad />

## Overview

Float ユーティリティ クラスは [CSS float property](https://developer.mozilla.org/ja/docs/Web/CSS/float)を使用し、現在のビューポート サイズに基づいてフローティングを適用します。
